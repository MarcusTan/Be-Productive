# Be Productive
- Time interval tracking utility.
- s = start, c = cancel, enter = end
- localstorage allows persistence while refreshing page

## Setup
- `npm install`
- `node index.js`
- run `nohup node index.js >/dev/null 2>&1 &` to run in the background
